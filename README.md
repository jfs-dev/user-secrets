## Sobre o projeto
Este projeto mostra como gerenciar informações sensíveis com User Secrets em C#.

Em C#, <code>user-secrets</code> é uma funcionalidade muito útil para gerenciar informações sensíveis, como chaves de API, strings de conexão e outras configurações que não devem ser incluídas no controle de versão. Esta funcionalidade é especialmente útil durante o desenvolvimento.

Essa abordagem garante que suas informações sensíveis permaneçam seguras e fora do controle de versão, enquanto são facilmente acessadas durante o desenvolvimento.
